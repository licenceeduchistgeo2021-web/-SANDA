# **App Name**: SANDA Audit Tool

## Core Features:

- CSV Data Extraction: Extract all 80 questions and associated options (L1 to L5) from the uploaded CSV file.
- Axis-Based Question Display: Display questions grouped by their corresponding axis ('المحور') as specified in the CSV: Understanding, Governance, Investment, and Readiness.
- Compulsory Governorate Selection: Require the user to select their Governorate (عمالة) from a dropdown list (Select Menu) before starting the survey.
- Radio Button Question Options: Present each question with five radio button options (L1 to L5) derived from the CSV data.
- Axis Score Calculation: Automatically calculate the total score for each axis based on user responses.
- Radar Chart Visualization: Display a radar chart illustrating the progress level for each Governorate based on aggregated axis scores.
- PDF Report Export with Governorate Name: Generate a PDF report containing the audit results, clearly indicating the selected Governorate name in the report header.

## Style Guidelines:

- Primary color: Deep blue (#1A5276) to evoke trust and professionalism. This aligns with the serious nature of auditing and governance.
- Background color: Very light blue (#EBF5FB), close in hue to the primary color but highly desaturated and bright, for a clean, uncluttered backdrop.
- Accent color: Burnt orange (#D35400) to create contrast, for call-to-action buttons (like 'Export PDF') and key indicators on the radar chart.
- Body and headline font: 'Inter', a grotesque-style sans-serif font. Suitable for both headlines and body text.
- Use a clean, well-organized layout to present questions, options, and results clearly. Group questions by axis using distinct sections or cards.
- Incorporate subtle icons to represent each axis (Understanding, Governance, Investment, Readiness) for visual clarity.
- Use smooth transitions and animations to enhance the user experience, especially when switching between axes or generating the radar chart.
